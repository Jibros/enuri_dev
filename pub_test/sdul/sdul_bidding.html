입찰규칙
  • ① 입찰대상
    에누리에서 정상 서비스되고 있는 상품
    ② 입찰단가 & 일노출수 제한
    입찰 건당 최소 노출 단가는 10원이며, 10원 단위로 증액할 수 있음
    일 노출수 제한 값을 설정했을 경우, 설정한 값에 도달하면 입찰건은 자동으로 종료
    ③ 과금
    좌/우, 1/2/3 순위 입찰건에 대해 가격비교창이 노출될 시 과금
    일 노출수 제한 값을 설정했을 경우, 설정한 값에 도달하면 입찰건은 자동으로 종료
  • ④ 입찰대기
    입찰 건이 4순위 이하이거나 기타 변동 사항이 있어 상위에 노출되지 않는 경우 과금이 되지 않으며, 동일 최저가 4순위일 경우, 1,2,3 순위의 입찰 건이 종료되면 자동으로 상위에 노출
    ⑥ 예약입찰
    예약 입찰한 상품이 최저가가 되어 동일한 최저가와 경쟁하게 될 경우 과금
    예약 후에도 최저가가 아니거나 경쟁할 다른 최저가가 없다면 과금되지 않음
    ⑦ 찜, 관심상품, 삭제
    상품명의 체크박스를 클릭하면 찜, 관심상품, 삭제 기능을 사용할 수 있습니다.
  • ※ 이런 경우엔 과금이 되지 않습니다
    • - 상위 노출 도중 최저가 상품이 하나만 남는 경우 (단독 최저가)
    • - 내 판매가보다 낮은 최저가가 서비스될 경우 (판매가 밀림)
    • - 내 입찰단가 보다 더 높은 입찰단가가 있는 경우 (입찰가 밀림)
    • - 입찰 적용 중인 상품이 정상 서비스되지 않은 경우
    ※ 중지된 서비스는 대기 상태로 전환되며, 원인이 없어지면 자동으로 재시작

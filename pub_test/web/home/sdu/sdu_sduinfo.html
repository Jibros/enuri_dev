에누리 SDU사용안내
  • ○ 에누리™ 가격비교 상품 등록 과정
  • 1. 에누리에서는 상품정보 및 가격정보에 대해 1일 3~7회의 업데이트를 실시하고 있습니다.
  • 2. 등록된 상품은 "SDU>상품리스트>등록 현황>등록 상품수>보기"에서 확인하실 수 있습니다.
  • 검색 프로그램이 업데이트해서 가져온 상품 중 아직 매칭작업이 안된 상품은"SDU>상품리스트>등록 대기중 상품"에 나옵니다.
  • 3. "등록 대기중 상품"에 있는 자료는 아래와 같이 처리됩니다.
  • - 작업 진행중 : 1~3일 소요
  • - 미취급 상품 : 매칭이 안되고 대기중 리스트에 남게되며 에누리 사이트에서 쇼핑몰 검색으로 서비스 됨
  • - 에누리에 아직 없는 신규모델 : SDU로그인→상단대분류클릭→[신규모델등록요청] 메뉴를 이용하여 신청
  • - 등록보류 상품(상품의 이미지나 스펙 오류) : 오류 수정 후 다시 등록 요청
  • 4. 상품 등록 요청전 아래 사항을 확인하시기 바랍니다.
  • ○ 자료 출처가 검색페이지라면 요청하실 상품이 검색페이지에 있나요?
  • ● 아직 없다면 검색페이지에 상품을 넣은 후 요청해주세요.
  • (검색페이지 확인은 SDU로그인→상품리스트>세팅된 카테고리 클릭)
  • ○ 해당 상품이 품절처리 되어 있지는 않나요?
  • ● 품절 상품은 서비스가 안됩니다.
  • ○ 새로 추가된 카테고리에 있는 상품을 요청하시나요?
  • ● 카테고리 추가나 변경시 SDU회원게시판에 변경사항을 미리 알려주세요.
  • ○ 해당 상품의 가격이 다른 쇼핑몰과 비교하여 너무 낮거나 높지 않나요?
  • ● 상품가격을 확인해주세요.
  • (0 하나가 빠져있거나 더 추가되어 있는지 점검해주세요.)
  • ○ 해당 상품에 상품 이미지와 상세정보는 정확한가요?
  • ● 이미지나 스펙이 없거나 틀린 경우 등록이 보류됩니다.
  • ○ 서비스 제품수가 많이 모자란가요?
  • ● 에누리에서 바로 서비스 할 수 있는 상품으로 자료를 더 늘려주세요.
  • * 근무시간(월~금) : 09:00 ~ 18:00
  • * 점심시간 : 11:40 ~ 12:40
  • ※ 전화는 수백개 협력사의 요청처리에 충분한 수단이 못 되니
  • 효율적인 업무진행을 위해 SDU회원게시판을 이용해주십시오.

    ○ 에누리 가격비교 상품 관리

    1. 에누리의 등록된 모든 상품은 가격수정, 품절처리, 이벤트 설정, 배송정보, 옵션설정(안심구매) 등의 입력이 가능합니다.
     - 가격수정 : 에누리 검색프로그램이 쇼핑몰 홈페이지에 업데이트 할 때 자동으로 수정되지만 SDU에서 직접 수정할 경우 실시간으로 적용됩니다
     - 품절처리 : 해당 상품을 "품절"처리 하거나 "삭제"처리 할 경우 에누리 서비스에서 제외되며, 전체 상품 수량에도 포함되지 않습니다.(추후 "서비스예외 상품" 모드로 일괄 확인 가능)
     - 이벤트 설정 : 개별, 카테고리별, 전체 상품의 이벤트 설정이 가능합니다.

    ○ 에누리 상품 검색 방법

    1. 에누리용 검색페이지에서 검색: 서비스 할 상품 리스트를 HTML로 이루어진 별도의 페이지에 제작하여 자체 서버에 올려놓아 에누리에서 검색이 가능하도록 합니다.
     - 쇼핑몰의 모델명, 가격, 재고정보가 실시간으로 반영되도록 ASP, PHP 등의 프로그램을 사용하여 제작해야 합니다.

    2. 쇼핑몰에서 직접 검색 : 별도의 페이지를 제작하기 어려운 쇼핑몰이 사용하며, 에누리 검색프로그램이 쇼핑몰 홈페이지에 직접 접속하여 상품을 검색합니다.
     - 상품 리스트 페이지에 모델명, 가격, 품절 정보가 있어야 합니다.
     - 상품 리스트 페이지의 URL이 규칙적이어야 합니다.
     - 카테고리(대분류, 중분류, 소분류, 미분류)를 지정하여 상품수량 조절이 가능해야 합니다.

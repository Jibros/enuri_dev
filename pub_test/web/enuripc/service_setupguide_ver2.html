MS 윈도우 10 설치하기 (MS Windows 10 Install)

PC 사용을 위해서는 보편적으로는 윈도우를 설치 해야하며, 아래에서 설치하는 과정을 참고하시길 바랍니다.

우선 윈도우10을 설치하기 위해서는 윈도우10 정품(키) 와 USB(8GB 이상) 가 필요 합니다. 윈도우10 정품은 에누리사이트 아래 링크에서 구입이 가능하며 에누리 조립PC는 윈도우즈 정품사용을 권장합니다. 윈도우10은 보편적으로 HOME, PRO, FPP(처음사용자용), DSP(32bit, 64bit), ESD 등으로 나누어 판매되고 있으며, 아래 설치과정은 위 나열된 어떠한 버전을 구매하셔도 공통적으로 적용되는 과정 입니다. 윈도우10 정품을 구입하시면 윈도우10 정품 시리얼 키를 받게 되며 설치 시 이 키를 넣어 정품인증 및 업데이트를 받을 수 있습니다. FPP타입을 구입하게되면 윈도우가 탑재된 USB를 제공받게 되는데 해당 USB로 윈도우를 설치하셔도 되지만, 제공 받는 USB에 윈도우는 최신버전이 아닐수 있으므로 마이크로소프트 사이트의 윈도우10 다운로드 페이지로 이동한 뒤 최신버전을 받아서 설치하시면 더욱 쾌적한 설치 및 사용환경이 되실 수 있으므로 사이트 방문을 권장합니다.

마이크로소프트는 정품 윈도우를 자유롭게 다운로드 할 수 있게 해 두었으며, 윈도우 시리얼키를 구입한 뒤 설치 중 인증을 받는 방법으로 설치 할 수 있습니다. 이러한 방식으로 다운로드된 윈도우를 어둠의 경로에서 흔히들 이야기하는 MSDN 버전이라 불렀습니다. MICROSOFT DOWN 의 줄임말이였죠. 해당 링크를 통해 윈도우 10 설치시 최신버전의 업데이트가 완료된 윈도우10 이 설치되며(현재 레드스톤4 1803) 추후 업데이트시에는 최신버전으로 바뀌게 됩니다. 다운로드 페이지에서 윈도우 설치용 USB 만들기를 진행하면 하단과 같은 화면순서대로 진행됩니다.

  1. 윈도우 설치프로그램을 인스톨할 USB를 삽입 후 페이지내 지금 도구 다운로드를 클릭하면 윈도우 설치용 USB 제작을 진행할 수 있습니다.
  2. 소프트웨어 사용권 계약서가 나오는데 동의를 클릭합니다.
  3. 미디어(USB, DVD)에 설치 프로그램을 만들수 있게 화면이 나오며 다른 PC용 설치미디어 만들기를 선택한 뒤 다음으로 진행합니다.
  4. PC권장옵션 사용 체크를 해제하면 아키텍처 에서 32비트, 64비트, 모두를 선택할수 있으며, 이왕이면 모두를 선택 후 진행합니다.
  5. 윈도우 설치 시 버전에 맞는 제품키가 필요하다는 안내가 나오며 확인을 클릭하여 다음으로 진행합니다.
  6. 설치할 미디어를 선택할 수 있으며 USB를 선택하여 진행합니다.
  7. USB의 경로를 설정하는 것으로 설치과정 전 이미 삽입한 USB드라이브가 지정되었으며 다음으로 진행하면 됩니다.
  8. 다운로드가 완료되면 다운로드를 확인하는 과정으로 넘어갑니다.
  9. 다운로드 확인 중
  10. 확인 후 미디어 제작을 진행 합니다.
  11. USB설치미디어가 완성 되었습니다. 마침을 클릭하면 다운로드 된 프로그램을 정리하며 마무리 됩니다.

새 PC에 윈도우10 설치하기

윈도우10을 PC에 설치하기 위해서는 윈도우10 설치 프로그램이 담긴 USB 나 DVD 미디어를 윈도우를 설치할 PC에 넣고 설치를 진행하시게 됩니다.
  • Step 1. 바이오스 설정
  • Step 2. 제품키 등록 및 파티션 설정
  • Step 3. 재부팅 후 설정

바이오스 설정

우선 부팅시 윈도우 미디어가 담긴 USB나 DVD를 우선 부팅순서로 바꾸는 바이오스 세팅을 해주어야 합니다. (2가지 방법이 존재하며 각각 아래 설명하겠습니다.)

아래 화면이 나왔을때 (GIGABYTE 메인보드 기준으로 설명 ASUS, ASROCK 등 보드도 크게 다르지 않음)

BIOS SETUP 으로 진입한 경우

DEL 키를 눌러 BIOS 세팅 메뉴로 들어가서 부팅순서를 바꾸는 방법

  1. 3번재 BIOS(바이오스) 메뉴로 가면 Boot Option 이 있는데 #1, #2 순서 중 1번에 윈도우 설치가 담긴 USB나 DVD로 세팅하면 됩니다.
  2. USB 인 Lexar USB 로 선택하여 부팅순서 1번에 USB 드라이브가 입력되면 완료되며 상단 메뉴 중 Save & Exit 메뉴로 이동하여 Save & Exit 를 선택하면 재부팅 되며 USB 미디어로 부팅이 진행됩니다.
F12 BOOT MENU(부트 메뉴)로 진입한 경우

F12 키를 눌러 부팅순서를 바꾸는 방법 (타 회사의 메인보드는 F12 가 아닐수 있음)

  1. 위 화면과 같이 부팅할 드라이브나 미디어를 선택 할 수 있는 화면이 뜹니다. 이중 Lexar USB 를 선택하면 USB 미디어로 부팅이 진행됩니다. 위 방법으로 USB 미디어로 부팅을 진행하면 아래와 같은 화면이 나오며 이미지 순서대로 진행하면 됩니다.
설치방법
  1. 사용하실 컴퓨터의 운영체제를 32Bit, 64Bit 에 맞게 선택하여 설치하시면 되며 컴퓨터 메모리가 4GB 이상이면 64Bit 로 설치하셔야 됩니다. (USB에 윈도우를 담을때 32Bit, 64Bit 를 선택하셨으면 위화면은 노출되지 않고 바로 우측 화면으로 진행됩니다.)
  2. 위 화면들이 지나가면 보라색 윈도우 설치 화면이 노출 됩니다.
  3. 언어 및 키보드 선택 옵션 화면이며 한국어가 기본으로 선택되어 있으며 키보드도 PC/AT 101키 호환 키보드가 기본으로 선택되어 있어크게 변경하지 않고 다음으로 넘어가시면 됩니다.
  4. 지금 설치를 선택

제품키 등록 및 파티션 설정

  1. 윈도우 정품 인증 화면이 나오며 해당 화면에 구매하신 정품 윈도우 시리얼키를 입력하시면 됩니다. 정품키를 정확히 입력하시면 오른쪽 아래 다음 버튼이 활성화 되어 다음을 클릭하 실수 있습니다. 추후 윈도우를 구매셔서 인증 하실 예정이시거나 추후 입력하시길 윈하시는 경우에는 “제품키가 없음” 을 클릭하셔서 진행하셔도됩니다. 윈도우 설치후 정품인증이 가능하시기 때문에 해당 과정을 넘어가셔도 윈도우 설치에는 지장이 없습니다
  2. 소프트웨어 사용권 동의에 대한 화면이며 동의함을 체크하고 다음을 클릭
  3. 위 화면도 윈도우 사용에 대한 동의 화면이며 동의함 체크 후 다음을 클릭 윈도우 버전 및 보유 라이센스에 따라 위 화면이 노출되지 않고 다음 화면이 바로 나오는 경우도 있습니다.
  4. 윈도우 설치유형 선택 과정이며 최초 윈도우를 설치하는 경우이므로 “ 사용자지정 Windows만 설치(고급) “ 를 선택합니다.
  5. 아직 윈도우가 설치되지 않은 하드디스크(SSD) 드라이브를 선택하는 화면이 노출 됩니다. 해당 디스크는 파티션 설정과 포맷이 되지 않은 상태로 할당되지 않은 공간으로 표시되고 있습니다. 긴장하지 말고 새로만들기 를 클릭하시면 아래 화면이 나옵니다. 순서대로 진행하시면 됩니다
  6. 새로만들기를 클릭하시면 파티션 크기를 지정하는 내용이 추가되는데 해당 숫자를 수정하지 않고 그대로 적용을 클릭하시면 됩니다.
    1. 각 PC에 설치된 하드디스크(SSD) 용량에 따라 다른 숫자가 나올수 있습니다. 예시는 120GB SSD 디스크를 사용하였으며 240GB, 512GB, 1TB, 2TB 용량의 디스크가 설치된 경우에는 다른 숫자가 나오며 해당디스크 전체크기에 대한 숫자가 노출될 것입니다. 그대로 적용을 클릭하시면 됩니다.
    2. 새로만들기 → 크기 내 숫자를 건들이지 않고 적용으로 진행하시면 됩니다.
    3. 확인을 클릭 하면 우측의 두가지 화면 중 하나가 노출 될 것입니다.
    4. 두 화면다 모두 윈도우 설치 과정 중 모두 나타날수 있으며 윈도우 설치에 문제가 없습니다. 위 두가지 화면이 파티션 작업이 끝난 후 화면이며 종류 중 주로 표기된 주파티션에 윈도우를 설치하면 됩니다. 다음을 클릭하면 설치중 화면들이 순서대로 노출되며 자동으로 윈도우를 설치가 됩니다.

재부팅 후 설정

[설치중 화면] 순서대로 진행된 후 자동 재부팅 됩니다.

재부팅시 처음 BIOS 에서 윈도우 설치 USB 를 Boot Option #1 으로 해놓으시면 최초 윈도우 설치 화면이 나올수 있습니다. 이때는 USB 를 제거 하시고 ESC 키를 누르시면 됩니다.

[재부팅 화면] 장치 설정완료 후 자동 재부팅이 진행됩니다.

설정 방법
  1. 위 화면이 나오면 한국으로 자동 설정되어 나오며 지역을 “예” 를 클릭 다음으로 진행합니다.
  2. 한글입력기 설정화면도 “예”를 클릭 다음으로 진행합니다.
  3. 키보드 레이아웃 설정이 나오며 특별히 두번째 키보드를 사용하지 않는다면 “건너뛰기” 로 다음으로 진행합니다.
  4. 네트워크 설정도 윈도우 설치 완료 후 네트워크 드라이버 등을 설치하기 위해 건너뛰기로 넘어갑니다.
  5. 사용하실 사용자의 이름을 넣습니다. 추후 변경이 되므로 사용자 이름을 너무 고민하실 필요는 없습니다.
  6. 패스워드를 설정합니다. 이것도 추후 변경가능하므로 쉬운 비밀 번호를 우선 설정하셔도 무방합니다.
  7. 개인 정보 설명 메뉴로 수락으로 다음으로 진행합니다. 수락하면 설치중 화면이 출력되며 순서대로 자동 진행됩니다.
  8. 설치가 완료되면 윈도우 화면으로 출력됩니다.

드라이버 설치 가이드

Windows 10은 설치과정에서 일반적으로 드라이버 설치가 되지만, 네트워크 연결이 되지 않을 때, 혹은 기타 유사시를 대비하여 수동으로 설치하는 방법을 소개합니다.
  • Step 1. PC 바탕화면 설정하기
  • Step 2. 드라이버 설치하기
  • Step 3. Windows 업데이트

잠깐, 드라이버 설정 이전에 간단한 설정부터 하고 진행하겠습니다.

  1. Windows 10 설치가 완료되었지만, 뭔가 허전합니다. 따라서, ‘내 컴퓨터’와 ‘내 문서’ 같이 자주 쓰는 폴더를 바탕화면에 보이도록 하겠습니다.
  2. 빈 공간에 마우스 포인터를 위치시키고, 우측버튼을 클릭! 메뉴창이 나타나면, ‘개인설정(R)’을 클릭합니다.
  3. 왼쪽 메뉴에서 ‘테마’를 선택합니다.
  4. 오른쪽 화면에 마우스 스크롤을 내려서 ‘관련설정’ 항목에 ‘바탕 화면 아이콘 설정‘을 클릭합니다.
  5. 팝업창이 뜨고 체크박스에서 필요한 아이콘에 체크한 뒤, ‘확인‘을 클릭합니다.
  6. 바탕화면에 ‘내 PC’, ‘네트워크‘, 제어판’ 등의 아이콘들이 생성되었습니다. 이어서 ‘Internet Explorer’를 바탕화면에 보이도록 설정해 보겠습니다.
  7. 화면의 좌측하단 Windows 로고를 클릭하여 시작메뉴를 열면, 우측 목록에 ‘Windows 보조프로그램’이 나타나며, 하위 메뉴에 ‘Internet Explorer’가 있습니다. 그 위에 마우스 오버를 한 뒤, 우측버튼을 클릭합니다.
  8. ‘자세히’ 옆의 ‘>’ 표시를 클릭하면, 메뉴가 보이게 됩니다. 여기서 ‘작업 표시줄에 고정’을 클릭합니다..
  9. 작업 표시줄 가장 오른쪽에 ‘Internet Explorer’ 아이콘이 생성되었습니다.
PC에 DVD가 없으면 제조사 홈페이지에서 드라이버를 다운로드할 수 있습니다.

‘Windows 10’ 설치가 완료되면 드라이버 설치를 해야 합니다. 드라이버는 메인보드에 동봉된 CD나 메인보드 제조사의 홈페이지에서 다운로드 받아서 설치가 가능합니다. Windows가 정상적인 구동을 할 수 있도록 처음 1번은 세팅해야 차후에 설치되는 드라이버를 자동으로 잡을 수 있습니다. 메인보드 구매시 동봉된 CD가 없다면, 하단의 링크에서 각 제조사별 드라이버를 다운로드 받을 수 있습니다.

만약, 네트워크 연결이 되지 않을 때, 3DP Net을 통해 드라이버를 설정합니다. 3DP Net은 이더넷(랜) 카드를 자동으로 인식하고 최적의 드라이버를 연결해 주는 프로그램입니다. USB를 통해 다른 컴퓨터에서 3DP Net을 다운로드 받은 뒤 복사합니다. (네트워크 연결이 완료된 경우, 5번으로 이동합니다.)

  1. 사전에 복사해 둔 설치 프로그램을 실행하며, 설치 경로를 지정해주고, ‘OK’를 클릭 합니다.
  2. 위와 같은 화면이 나오면 설치가 진행되고 있는 것 입니다.
  3. 설치가 완료되면 3DP_Net이 실행되며, 하단의 오렌지색 글씨로 적힌 ‘랜 카드’ 항목을 더블 클릭하여 실행합니다.
  4. 위와 같이 드라이버 설치 프로그램이 실행되며, ‘다음’을 선택하여 설치를 진행합니다.
  5. ‘마침’을 클릭하여, 네트워크 드라이버 설정을 완료합니다.인터넷이 연결되면, ‘3DP Chip’이라는 프로그램을 통해 PC를 최신 또는 최적의 드라이버 상태로 설정합니다.
    6. ‘3DP Chip’의 다운로드 방법은 ‘3DP Net‘과 동일하며, 아래의 링크로 접속하면 다운로드 가능합니다.
  6. ‘최신 버전 다운로드는 이곳을 클릭하세요’를 클릭하여, 파일을 다운로드 받아 설치합니다.
  7. ‘다음’을 클릭하여 다음단계로 진행합니다. 구성요소, 설치위치 등을 설정해 주면, 프로그램 설치가 시작됩니다.
  8. 설치되는 과정에서 몇 차례 애드웨어가 발생하는데, 이는 필요에 따라 체크해제하고 다음을 클릭하면 됩니다. (설치가 진행하는 동안 Himoum, Avast Free Antivirus, Opera 등이 나왔습니다.)
  9. 위와 같이 프로그램이 설치가 되고 있습니다.
  10. 프로그램이 실행되면 위와 같은 리스트가 나타나고, 각 항목별로 최신 드라이버를 설정할 수 있습니다.
  11. 위와 같이 최신 드라이버를 설치할 수 있도록 인터넷으로 접속되며, ‘드라이버 다운로드’를 클릭하여 다운로드 후 설치하면 됩니다.

하단의 이미지를 클릭하면 제조사 다운로드 페이지가 연결됩니다.

  1. 이제부터 각각의 소프트웨어를 설치해 보겠습니다. 우선 백신 항목의 V3 Lite 링크에서 프로그램을 다운로드 받아 실행합니다. 파일을 실행하고, ‘예’를 선택한 뒤, 설치 창이 나타나면 ‘다음’을 클릭합니다.
  2. 사용권 계약 및 데이터 수집 동의에 관한 사항은 확인 후, 체크박스에 체크를 하고, ‘다음’을 클릭합니다.
  3. 설치할 폴더를 지정한 뒤, ‘설치’ 버튼을 클릭하면, 위와 같이 프로그램이 설치됩니다.
  4. 설치가 끝나면 위와 같은 화면이 나타나며, ‘마침’ 버튼을 클릭합니다. (체크박스는 필요에 따라 체크를 하거나, 하지 않으셔도 무방합니다.)
  5. 이번엔 알집을 설치해 보겠습니다. 절차는 ‘V3 Lite’와 비슷하며, 프로그램을 다운로드 받아 실행합니다. 파일을 실행하고, ‘예’를 선택합니다.
  6. 저장할 위치를 지정해주고, 동의서 내용을 확인하여 이상이 없다면, ‘동의’ 버튼을 클릭합니다.
  7. ‘설치 시작(N)’ 버튼을 클릭합니다. (하단의 체크박스는 필요에 따라 체크를 하고, 체크를 하지 않으셔도 무방합니다.)
  8. ‘빠른 설치(N)’을 클릭합니다. (이번에도 하단의 체크박스는 필요에 따라 체크를 하고, 체크를 하지 않으셔도 무방합니다.)
  9. 설치가 진행되는 중입니다.
  10. 설치가 완료되었을 때, 나타나는 화면입니다. 하단의 ‘알약’은 ‘V3 Lite’가 있으므로 체크를 해제하고 ‘확인’을 클릭합니다.
  11. 이번에는 동영상 플레이어를 설치해 보겠습니다. 절차는 ‘V3 Lite’와 비슷하며, 다운로드 후 실행합니다. 파일 실행 후, ‘예’를 선택한 뒤, 저장할 위치를 지정합니다. 설치에 사용될 언어에 ‘Korean’ 인지 확인 후, 맞다면 ‘확인’을 클릭합니다.
  12. 동의서 내용을 확인 후, ‘동의합니다(A)’ 탭을 선택한 뒤, ‘다음’ 버튼을 클릭하여 파일을 설치합니다.
  13. 절차는 ‘V3 Lite’와 비슷하며, 다운로드 후, 실행합니다. 파일을 실행 후, ‘예’를 선택한 뒤, ‘CHROME 다운로드’ 버튼을 클릭합니다.
  14. 서비스 약관을 확인 후, 왼쪽 아래에 있는 ‘동의 및 설치’ 버튼을 클릭합니다.
  15. 설치가 완료되면 위와 같은 이미지가 나타나며, 필요에 따라 기본 브라우저를 설정해 줍니다.
